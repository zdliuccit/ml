# Koa服务

## Package 说明
* koa # 主程
* koa-router # 路由
* koa-compress # 压缩响应
* koa-logger # 日志
* koa-session # session
* koa-static # 静态资源服务
* keygrip # 加密
* @babel/core # 支持es6语法
* @babel/preset-env # 支持es6语法
* @babel/register # 支持es6语法
* axios # http请求
* mongoose # Mongoose是设计用于异步环境的MongoDB对象建模工具
* nodemon # nodemon是一种工具，可在检测到目录中的文件更改时通过自动重新启动节点应用程序来帮助开发基于node.js的应用程序
